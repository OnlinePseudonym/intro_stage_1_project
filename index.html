Kevin's Notes

Web

The web is made up of HTML documents linked to each other via hyperlinks.
  • HTTP - main protocol of the web
  • Servers - computers that host the files that makeup the web
  • Internet - worlds largest computer network
  • Browser - Program that runs on your computer to display web docs

HTML

Hyper Text Markup Language
  • -text (content)
  • -markup (format?)
  • -references to other documents
  • -links to other pages

Tags

e.g. <name>contents</name>
  • Bold: <b>contents</b>
  • Italics: <em>contents</em>

HTML Attributes

e.g. <tag attr="value">contents</tag>
  • Ancher: <a href="www.reddit.com">derp</a>
  • Images: <img src="url" alt="text">

Whitespace

Any amount of whitespace turns into one empty space
  • Break: <br> (void)
  • Paragraph:<p>contents</p>

Containers

  • Span: <span>contents</span> (inline)
  • Div: <div>contents</div> (block)

HTML document structure:

  • <!DOCTYPE HTML>
  • <html>
    • <head>
      • <title>title</title>
    • <head>
    • <body>
      • contents
    • </body>
  • </html>
